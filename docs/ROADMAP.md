# Roadmap / 路线图

## Short Term / 近期

- Polish cinema room parity with the private reference app
- Improve playlist and companion-plan editing
- Tighten mobile and tablet layouts
- Improve import/export conflict handling
- Expand local retrieval diagnostics

## Medium Term / 中期

- Optional vector retrieval module
- Better memory-note organization
- PWA support
- Electron packaging guide
- More reusable component boundaries

## Long Term / 长期

- Full RAG pipeline
- Shared watch-room mode
- More provider adapters
- Desktop/mobile native shells
- Optional MCP hooks for media metadata and local notes

## Principle / 原则

Feature parity can grow over time, but private persona, private memory, and real chat history should stay user-owned.

功能可以逐步接近完整小屋，但私人人格、私人记忆和真实聊天历史应始终由用户自己掌握。
